## Condições Aninhadas

~~~~JavaScript
if(condicao1){
    bloco1
}else{
    if(condicao2){
        bloco2
    }else{
        bloco3
    }
}
~~~~

***

## Condição Múltipla

~~~JavaScript
switch(expressão){
    case valor 1:
        bloco1
        break
    case valor 2:
        bloco2
        break
    case valor 3:
        bloco3
        break
    default:
        bloco4
        break
}       
~~~~

**Observação:** Não esqueça de colocar os 'break'.
